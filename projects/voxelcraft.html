VoxelCraft

✓ Verified working — all source parses; documented build reproduces the page byte-for-byte

VoxelCraft is a browser-based, Minecraft-style voxel sandbox written in vanilla JavaScript and rendered with three.js. The entire game ships as a single self-contained HTML file of about 3,840 lines, authored as eight modular source parts that are concatenated into the final page. Notably it uses no image or audio asset files at all: every block and item texture is painted at runtime with Canvas 2D, and all sound effects are synthesized live through the Web Audio API.

Terrain is procedurally generated from seeded simplex noise across seven biomes with caves, ores and trees. The custom chunked renderer does face culling, ambient occlusion and smooth lighting via GLSL shaders, plus a day/night cycle with sun, moon, stars and fog. Gameplay includes survival and creative modes, a hotbar/inventory with 2x2 and 3x3 crafting, furnace smelting and chests, block mining and placement, eight mob types with AI, arrows, TNT and flowing water/lava — with player health, hunger, air, XP and armor, all persisted to localStorage.

Tech Stack

  • Vanilla JavaScript
  • three.js r128
  • WebGL / GLSL
  • HTML5 Canvas 2D
  • Web Audio API
  • localStorage
